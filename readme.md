# Azure Test Simple Static Website with Terraform

This project automates the deployment of a very simple, static HTML website on Azure Storage using Terraform.

## Features
* Automated Resource Group creation.
* Random name without separators generation for global uniqueness.
* Static website hosting on Azure Storage Account.

## How to run website
1. Initialize Terraform:`terraform init`
2. Plan configuration:  `terraform plan`
3. Apply configuration: `terraform apply`
